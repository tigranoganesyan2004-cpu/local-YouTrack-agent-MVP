# Локальный YouTrack Agent MVP

Локальный аналитический агент для работы с выгрузкой задач YouTrack.

Проект предназначен для быстрого анализа задач, поиска похожих кейсов, просмотра дедлайнов, фильтрации по полям и получения кратких объяснимых ответов по найденным данным. Агент работает локально, использует Ollama для генерации ответов и embeddings, FAISS для семантического поиска и SQLite для хранения истории запросов.

## Что умеет агент

На текущем этапе агент поддерживает:

- подготовку данных из XLSX/CSV выгрузки YouTrack;
- нормализацию структуры задач;
- удаление дублей по `issue_id`;
- группировку статусов в внутренние рабочие группы;
- lexical, semantic и hybrid retrieval;
- детерминированный поиск без LLM;
- retrieval + LLM по найденному контексту;
- поиск похожих задач;
- анализ новой постановки через найденные аналоги;
- фильтрацию задач по полям;
- подсчет по полям;
- поиск ближайших дедлайнов;
- локальное хранение истории запросов.

## Основная идея архитектуры

Агент работает по retrieval-first подходу:

1. сначала подготавливаются и нормализуются данные;
2. затем по задачам строится индекс;
3. пользователь задает запрос;
4. агент определяет режим;
5. выполняется точный, lexical, semantic или hybrid поиск;
6. при необходимости найденный контекст передается в LLM;
7. ответ возвращается в структурированном формате;
8. история сохраняется в SQLite.

Это позволяет отдельно контролировать:
- качество данных;
- качество retrieval;
- качество LLM-ответа;
- fallback-поведение при ошибках модели.

## Режимы работы

Агент поддерживает три пользовательских режима:

### 1. Авто
Агент сам выбирает оптимальный сценарий на основе запроса.

### 2. Точный поиск
Используется детерминированный режим без LLM. Полезен, когда нужен:
- воспроизводимый результат;
- поиск без генерации;
- проверка качества retrieval отдельно от модели.

### 3. Через LLM
Используется retrieval + LLM. Подходит для:
- кратких аналитических выводов;
- объяснения похожести задач;
- анализа новой формулировки через аналоги.
